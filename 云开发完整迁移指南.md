## 1. 云开发环境搭建步骤

### 1.1 云服务提供商选择

| 服务商 | 推荐理由 |
|--------|---------|
| **微信小程序云开发** | ✅ 原生支持，与小程序无缝集成<br>✅ 基础版免费，适合起步<br>✅ 开箱即用，无需额外配置<br>✅ 支持数据库、云函数、云存储、CDN |

### 1.2 账户注册与配置流程

#### 步骤 1：开通云开发
1. 打开微信开发者工具
2. 点击顶部菜单「云开发」
3. 点击「开通」，选择「按量付费」或「包年包月」
4. 选择环境名称（如 `photography-dev`）
5. 选择基础版（免费）即可

#### 步骤 2：创建环境
- **环境ID**：自动生成，记录下来
- **环境名称**：摄影预约（开发环境）
- **基础配置**：按量付费 → 基础版

### 1.3 必要的云服务资源创建

#### 1.3.1 数据库集合创建

| 集合名称 | 权限设置 | 用途 |
|---------|---------|------|
| `bookings` | 仅创建者可读写 | 预约订单 |
| `photos` | 所有用户可读，仅创建者可写 | 作品展示 |
| `notifications` | 仅创建者可读写 | 通知消息 |

**创建步骤**：
1. 进入云开发控制台 → 数据库
2. 点击「添加集合」
3. 输入集合名称
4. 设置权限（推荐：「仅创建者可读写」）

#### 1.3.2 云存储创建
- 云开发已默认开通云存储
- 存储路径建议：
  - `/orders/{orderId}/reference/` - 预约参考图
  - `/portfolio/{photoId}/` - 作品图片

#### 1.3.3 云函数创建
| 云函数名 | 用途 |
|---------|------|
| `createBooking` | 创建预约订单 |
| `updateBooking` | 更新订单 |
| `getBookings` | 获取订单列表 |

## 2. 迁移实施计划

### 2.1 现有本地数据迁移至云数据库

#### 迁移工具代码
创建 `utils/cloudMigration.js`：
```javascript
// 本地数据迁移到云开发
async function migrateLocalDataToCloud() {
  // 1. 读取本地订单
  const localOrders = wx.getStorageSync('photo_orders') || []
  
  // 2. 逐个上传到云数据库
  const db = wx.cloud.database()
  const collection = db.collection('bookings')
  
  for (const order of localOrders) {
    try {
      await collection.add({ data: order })
      console.log('迁移成功:', order._id)
    } catch (e) {
      console.error('迁移失败:', order._id, e)
    }
  }
  
  wx.showToast({ title: '迁移完成', icon: 'success' })
}

module.exports = { migrateLocalDataToCloud }
```

### 2.2 后端业务逻辑向云函数迁移

#### 云函数示例：`createBooking`
```javascript
const cloud = require('wx-server-sdk')
cloud.init({ env: cloud.DYNAMIC_CURRENT_ENV })
const db = cloud.database()

exports.main = async (event, context) => {
  const { OPENID } = cloud.getWXContext()
  const bookingData = event.booking
  
  try {
    const result = await db.collection('bookings').add({
      data: {
        ...bookingData,
        _openid: OPENID,
        createdAt: db.serverDate()
      }
    })
    
    return {
      success: true,
      orderId: result._id
    }
  } catch (e) {
    console.error(e)
    return {
      success: false,
      error: e.message
    }
  }
}
```

## 3. 性能优化方案

### 3.1 数据库查询优化建议

| 优化项 | 方案 |
|--------|------|
| **索引** | 为 `createdAt`、`status` 字段创建索引 |
| **分页** | 使用 `limit()` 和 `skip()` 分页加载 |
| **字段投影** | 只查询需要的字段 |
| **缓存** | 前端缓存最近的订单数据 |

**优化后的查询示例**：
```javascript
db.collection('bookings')
  .where({ _openid: '{openid}' })
  .field({ 
    _id: true, 
    date: true, 
    status: true,
    character: true
  })
  .orderBy('createdAt', 'desc')
  .limit(20)
  .get()
```

### 3.2 云函数执行效率提升方法

| 优化项 | 方案 |
|--------|------|
| **冷启动优化** | 云函数保持轻量，避免大依赖 |
| **数据库连接** | 复用数据库实例 |
| **并发处理** | 使用 Promise.all 并行操作 |
| **超时设置** | 根据业务调整超时时间 |

### 3.3 前端请求处理与状态管理优化

| 优化项 | 方案 |
|--------|------|
| **加载状态** | 提交时显示 loading，防止重复点击 |
| **乐观更新** | 先更新 UI，请求成功后再同步 |
| **错误重试** | 网络错误时自动重试（最多3次） |
| **防抖节流** | 频繁操作使用防抖 |

### 3.4 缓存策略实施建议

| 缓存内容 | 存储位置 | 有效期 |
|---------|---------|--------|
| 订单列表 | `wx.setStorage` | 5分钟 |
| 作品展示 | `wx.setStorage` | 30分钟 |
| 用户信息 | `wx.setStorage` | 永久（更新时刷新） |

## 4. 代码修改方案

### 4.1 创建云开发服务层

创建 `services/cloudService.js`：
```javascript
const db = wx.cloud.database()

// 创建订单
export async function createBooking(bookingData) {
  try {
    const res = await wx.cloud.callFunction({
      name: 'createBooking',
      data: { booking: bookingData }
    })
    return res.result
  } catch (e) {
    console.error('创建订单失败:', e)
    throw e
  }
}

// 获取订单列表
export async function getBookings() {
  try {
    const res = await db.collection('bookings')
      .where({ _openid: '{openid}' })
      .orderBy('createdAt', 'desc')
      .get()
    return res.data
  } catch (e) {
    console.error('获取订单失败:', e)
    throw e
  }
}

// 更新订单
export async function updateBooking(orderId, updateData) {
  try {
    const res = await db.collection('bookings')
      .doc(orderId)
      .update({ data: updateData })
    return res
  } catch (e) {
    console.error('更新订单失败:', e)
    throw e
  }
}
```

### 4.2 修改 booking-confirm 页面

修改 `pages/booking-confirm/booking-confirm.js`：
```javascript
import { createBooking } from '../../services/cloudService'

Page({
  async submitBooking() {
    if (this.data.isSubmitting) return
    
    this.setData({ isSubmitting: true })
    
    try {
      // 方式1：使用云函数（推荐）
      const result = await createBooking(this.data.booking)
      
      if (result.success) {
        wx.showToast({ title: '预约成功！', icon: 'success' })
        setTimeout(() => {
          wx.removeStorageSync('photo_booking_draft')
          wx.redirectTo({ url: '/pages/orders/orders' })
        }, 1000)
      }
    } catch (e) {
      console.error('提交失败:', e)
      wx.showToast({ title: '提交失败，请重试', icon: 'none' })
      this.setData({ isSubmitting: false })
    }
  }
})
```

### 4.3 修改 orders 页面

修改 `pages/orders/orders.js`：
```javascript
import { getBookings } from '../../services/cloudService'

Page({
  async loadOrders() {
    this.setData({ loading: true })
    
    try {
      // 方式1：从云数据库读取

      wx.showToast({ title: '加载失败', icon: 'none' })
      this.setData({ loading: false })
    }
  }
})
```
