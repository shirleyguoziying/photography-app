# 云开发使用指南

## 一、初始化云开发环境

### 1. 开通云开发
1. 打开微信开发者工具
2. 点击顶部工具栏「云开发」按钮
3. 点击「开通」
4. 选择基础版（免费）
5. 创建环境（环境名称可以是 `photography-app-dev`）
6. 记录环境 ID（格式类似：`cloud1-7gpg93cufb3a4583`）

### 2. 配置环境 ID
在 `app.js` 中添加云开发初始化：

```javascript
App({
  onLaunch() {
    wx.cloud.init({
      env: 'your-env-id-here', // 替换为你的环境ID
      traceUser: true
    })
  }
})
```

## 二、创建数据库集合

### 1. 打开云开发控制台
1. 点击顶部「云开发」按钮
2. 进入「数据库」标签

### 2. 创建以下集合

| 集合名称 | 说明 | 权限设置 |
|---------|------|---------|
| `bookings` | 预约订单 | 仅创建者可读写 |
| `orders` | 订单（同bookings） | 仅创建者可读写 |
| `photos` | 客片照片 | 所有用户可读，仅创建者可写 |
| `props` | 道具列表 | 所有用户可读，仅管理员可写 |
| `users` | 用户信息 | 仅创建者可读写 |

### 3. 设置权限
对于每个集合：
1. 点击集合名称
2. 点击「权限设置」
3. 选择对应的权限规则
4. 点击「确定」

## 四、增删改查操作示例

### 1. 添加数据（Create）
```javascript
// 添加预约
const db = wx.cloud.database()
db.collection('bookings').add({
  data: {
    ...formData,
    status: 'pending',
    createdAt: new Date()
  },
  success: res => {
    console.log('添加成功', res._id)
  }
})
```

### 2. 查询数据（Read）
```javascript
// 查询当前用户的所有预约
const db = wx.cloud.database()
db.collection('bookings')
  .where({
    _openid: '{openid}' // 自动获取当前用户
  })
  .orderBy('createdAt', 'desc')
  .get({
    success: res => {
      console.log('查询结果', res.data)
    }
  })

// 查询单个订单
db.collection('bookings').doc('order-id').get({
  success: res => {
    console.log('订单详情', res.data)
  }
})
```

### 3. 更新数据（Update）
```javascript
// 更新订单状态
const db = wx.cloud.database()
db.collection('bookings').doc('order-id').update({
  data: {
    status: 'confirmed',
    updatedAt: new Date()
  },
  success: res => {
    console.log('更新成功')
  }
})
```

### 4. 删除数据（Delete）
```javascript
// 删除订单
const db = wx.cloud.database()
db.collection('bookings').doc('order-id').remove({
  success: res => {
    console.log('删除成功')
  }
})
```

## 五、云存储使用

### 1. 上传图片
```javascript
wx.chooseImage({
  count: 1,
  success: res => {
    const tempFilePath = res.tempFilePaths[0]
    const cloudPath = 'images/' + Date.now() + '.jpg'
    
    wx.cloud.uploadFile({
      cloudPath: cloudPath,
      filePath: tempFilePath,
      success: res => {
        console.log('上传成功', res.fileID)
      }
    })
  }
})
```

### 2. 下载/预览图片
```javascript
// 获取临时链接
wx.cloud.getTempFileURL({
  fileList: ['cloud://your-env-id/xxx.jpg'],
  success: res => {
    const tempUrl = res.fileList[0].tempFileURL
    wx.previewImage({ urls: [tempUrl] })
  }
})
```

### 3. 删除文件
```javascript
wx.cloud.deleteFile({
  fileList: ['cloud://your-env-id/xxx.jpg'],
  success: res => {
    console.log('删除成功')
  }
})
```

## 六、权限控制

### 1. 数据库权限规则
- **仅创建者可读写**：适用于 bookings、orders、users
- **所有用户可读，仅创建者可写**：适用于 photos
- **所有用户可读，仅管理员可写**：适用于 props

### 2. 存储权限
- 默认：所有用户可读，仅上传者可写
- 可在云开发控制台调整

### 3. 管理员权限
在数据库中创建 `admins` 集合，存储管理员的 openid：

```javascript
// admins 集合
{
  _id: "auto-generated",
  openid: "user-openid",
  name: "管理员名称",
  createdAt: new Date()
}
```

## 七、共享访问配置

### 1. 数据共享
- 确保集合权限设置正确
- 对于 props 等公共数据，设置为「所有用户可读」

### 2. 云函数（如需）
如果需要复杂的权限控制，可以创建云函数：

```javascript
// 云函数示例：获取所有订单（管理员）
const cloud = require('wx-server-sdk')
cloud.init()
const db = cloud.database()

exports.main = async (event, context) => {
  const { OPENID } = cloud.getWXContext()
  
  // 检查是否是管理员
  const adminRes = await db.collection('admins')
    .where({ openid: OPENID })
    .get()
  
  if (adminRes.data.length === 0) {
    return { success: false, message: '无权限' }
  }
  
  // 获取所有订单
  const ordersRes = await db.collection('bookings')
    .orderBy('createdAt', 'desc')
    .get()
  
  return { success: true, data: ordersRes.data }
}
```

## 八、部署云函数

### 1. 创建云函数目录
在项目根目录创建 `cloudfunctions` 文件夹

### 2. 创建云函数
例如创建 `getBookings` 云函数：

```
cloudfunctions/
└── getBookings/
    ├── index.js
    ├── package.json
    └── config.json
```

### 3. 上传部署
1. 右键云函数文件夹
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成

## 九、常见问题

### Q: 云开发收费吗？
A: 基础版免费，包含一定额度的存储、数据库和云函数调用。

### Q: 如何切换环境？
A: 在 `app.js` 中修改 `env` 参数即可。

### Q: 数据安全如何保障？
A: 通过权限规则控制，确保用户只能访问自己的数据。

### Q: 可以备份数据吗？
A: 可以在云开发控制台导出数据库集合。
